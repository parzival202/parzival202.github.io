# Captures portfolio — Money Minder

Ces captures sont ordonnées pour présenter le projet **Money Minder** dans un portfolio ou une page projet.

## Présentation du projet

**Money Minder** est une application de gestion de finances personnelles. Elle permet de suivre les dépenses, les budgets, les KPIs financiers, l’épargne, les dettes et les alertes. Le projet est aussi pensé comme un garde-fou comportemental avec des indications visuelles et des notifications Telegram.

Stack à présenter dans le portfolio :

- PHP
- SQLite
- JavaScript
- Bootstrap
- Chart.js
- Telegram Bot API

## Workflow conseillé pour la page projet

1. **Vue globale** : montrer le tableau de bord avec les KPIs.
2. **Analyse des dépenses** : montrer la répartition, les dépenses récentes et la consommation du budget.
3. **Gestion budgétaire** : expliquer les budgets par catégorie et le suivi des seuils.
4. **Historique** : montrer la traçabilité des dépenses.
5. **Rapports** : montrer les graphiques et l’évolution des dépenses.
6. **Épargne** : montrer le suivi du taux d’épargne et des objectifs.
7. **Dettes** : montrer le suivi des dettes et remboursements.
8. **Alertes** : montrer la logique de notification et de contrôle des dépenses.

## Texte court utilisable dans le portfolio

Money Minder est une application de gestion budgétaire personnelle conçue pour aider à mieux contrôler les dépenses au quotidien. Elle centralise les dépenses, budgets, KPIs, rapports, objectifs d’épargne, dettes et alertes dans une interface simple. Le projet intègre aussi un système d’alertes Telegram afin de signaler les dépassements ou comportements financiers à risque.
