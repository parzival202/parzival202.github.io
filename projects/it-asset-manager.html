Application metier / gestion de parc

IT Asset Manager / ITAM

Application web personnalisee pour centraliser l'inventaire informatique, suivre les maintenances, documenter les interventions et visualiser l'etat du parc.

Voir le depot GitHub
Stack
Next.js, React, JavaScript/TypeScript, CSS, API routes, scripts PowerShell
Metier
Inventaire, maintenances, interventions, utilisateurs, alertes, statistiques
Contexte
Outil adapte a un besoin de suivi de parc dans un environnement d'entreprise

Probleme resolu

Un parc informatique devient difficile a suivre quand les machines, les utilisateurs, les garanties, les maintenances et les interventions restent disperses entre fichiers, messages et memoire terrain.

Solution

L'application propose un socle ITAM dedie : fiches equipements, statuts, sites/services, utilisateurs assignes, historique d'interventions, agenda de maintenance et tableaux de bord.

Fonctionnalites deduites du code

  • Inventaire machines avec tag, type, statut, marque, modele, serie, OS, RAM et stockage.
  • Association a un site, service, utilisateur et poste.
  • Suivi achat, fournisseur, garantie, fin de vie et intervalle de maintenance.
  • Maintenances preventives, correctives, remplacements et deploiements.
  • Interventions avec date, duree, technicien, statut et details.
  • Dashboard avec repartition parc, maintenances, interventions et ratio preventif/correctif.

Workflow

  1. Ajouter ou consulter une machine.
  2. Suivre son statut, son affectation et ses garanties.
  3. Planifier une maintenance ou documenter une intervention.
  4. Centraliser les informations du parc et surveiller les alertes.

Resultats

Le projet montre la capacite a partir d'un besoin metier reel et a produire une interface operationnelle, orientee exploitation, avec donnees structurees et indicateurs utiles.

Limites et suites

Les suites logiques : durcir l'authentification, stabiliser les tests, documenter le modele de donnees, ameliorer les exports et fiabiliser le deploiement.
